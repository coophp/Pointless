Pointless
=========

### Description

Static blog generator

### Requirement

* PHP 5.3+

### Installation

#### Clone Pointless from github

	git clone git://github.com/scarwu/Pointless.git
	
#### Add this line to `.bashrc` or `.zshrc` file
	
	export PATH=$PATH:/path/to/Pointless/Bin/

#### Init Pointless

	poi init
	
#### Generate Sample Blog

	poi gen
